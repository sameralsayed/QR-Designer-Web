# QR Designer Web

Create stunning, fully customizable QR codes right in your browser — **free and beautiful**.

![QR Designer Preview](https://via.placeholder.com/800x400/667eea/ffffff?text=QR+Designer+Web)

## ✨ Features
- 🎨 **Beautiful customization** — colors, error correction, logos
- 📱 **Multiple content types** — URL, Text, Wi-Fi networks
- 📥 **Easy download** — PNG (high quality)
- 📲 **Fully responsive** — works perfectly on mobile, tablet, and desktop
- ⚡ **Fast & lightweight** — no backend, runs 100% in browser
- 🌐 **SEO friendly** — optimized meta tags and structure

## 🚀 How to Use
1. Clone or download the repo
2. Open `index.html` in your browser
3. Enter your content, customize colors/logo, and generate
4. Download your beautiful QR code!

## 🛠️ Technologies
- HTML5
- CSS3
- Bootstrap 5
- jQuery
- qrcode.js
